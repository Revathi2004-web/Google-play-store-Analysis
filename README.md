# Google Play Store Analysis

## Project Overview

This project focuses on analyzing Google Play Store applications and user reviews using Data Analytics techniques. The analysis includes app categories, ratings, installs, reviews, and user sentiments to understand app performance and user engagement.

---

## Dataset Information

The project uses two datasets:

* apps.csv
* user_reviews.csv

The datasets contain information related to:

* App names
* Categories
* Ratings
* Reviews
* Installs
* Free and Paid apps
* User sentiments
* Sentiment polarity

---

## Project Workflow

### Data Cleaning

* Removed duplicate records
* Handled missing values
* Converted rating column into numeric format
* Cleaned and prepared datasets for analysis

### Exploratory Data Analysis

* Top app categories analysis
* Ratings distribution analysis
* Free vs Paid apps comparison
* Most reviewed apps analysis

### Sentiment Analysis

* Positive sentiment analysis
* Negative sentiment analysis
* Neutral sentiment analysis
* Sentiment visualization using charts

---

## Visualizations

The project includes:

* Bar charts
* Histogram plots
* Pie charts
* Sentiment count plots

---

## Key Insights

* Most apps on the Play Store are free apps
* Some categories dominate the Play Store market
* User sentiment is mostly positive
* Ratings and reviews strongly influence app popularity
